FONT-SIZE PROPERTY

Description

The font-size property sets the size of a font.

Syntax

font-size : medium|xx-small|x-small|small|large|x-large|xx-large|smaller|larger|length|initial|inherit

Detailed View

Value Description
medium Sets the font-size to a medium size. This is default
xx-small Sets the font-size to an xx-small size
small Sets the font-size to small size
smaller Sets the font size to a smaller size than the parent element

Sample Code

Output

1. This fontsize is xx-small
2. This font-size is small
3. This font-size is smaller
4. This font-size is medium
5. This fontsize is large
6. This fontsize is x-large
7. This fontsize is xx-large
8. This fontsize is larger
9. This fontsize is inherit
10. This fontsize is 140%
11. This fontsize is 30px
